# ecommerce-node

a backend application focusing on using mysql to create and seed a database for various ecom products.

## Usage

open integrated terminal and use the following commands:

```
mysql -u root -p
(enter your password)
source db/schema.sql
quit
```

open new terminal:

```
npm run seed
npm run start
```
